HOW TO START

Guide on how to use:

  1. Choose Member or Guest at the kiosk
  2. Scan RFID (for members) or select vehicle type (for guests)
  3. Customize wash options.

    Select Vehicle (for guests) • Select type of soap: Basic & Premium

  4. Top-up or redeem points if balance is low (members)
  5. Tap the Start wash button to begin the carwash.
